# 代码位置 

目前有三个自动化流程

export_en.yaml:将文章转成英文 (手动执行）

update.yaml: 导出飞书文章（手动加自动：每天一次）

publish_github.yaml:将文章发布到github pages（手动加自动：有修改就触发）

具体语法，参考官方文档
