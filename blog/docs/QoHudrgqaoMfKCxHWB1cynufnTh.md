# Worker

worker可以让你利用cloudflare的服务器资源。和github action类似，但是比github action更灵活，因为他可以运行javascript和python.

https://developers.cloudflare.com/workers/get-started/guide/

## 通过命令行新建worker

### 新建

```yaml
npm create cloudflare@latest -- my-first-worker
```

### 运行

```yaml
npm run dev
```

### 部署

```yaml
npm run deploy
```
